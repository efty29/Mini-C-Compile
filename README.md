# Mini C Compiler

A complete C compiler built with C++ core, Node.js backend, and React frontend.

## Features

- ✅ Lexical Analysis (Tokenization)
- ✅ Syntax Analysis (AST Generation)
- ✅ Semantic Analysis (Type checking & Symbol Table)
- ✅ Code Generation (C code output)
- ✅ Beautiful React UI
- ✅ Real-time compilation

## Installation

```bash
# Clone or create project
npm run setup

# Start the application
npm start
